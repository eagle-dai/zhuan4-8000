# zhuan4-8000

英语专业四级（TEM-4）词汇书扫描页 → Anki 卡片。

把纸质词汇书拍照/扫描后，OCR 解析成结构化数据，再生成可导入 Anki 的记忆卡片，方便按词条背单词。

## 内容

- `pages/` —— 词汇书扫描原件（JPG）。持续增量补充。
- `ocr/` —— 每页解析出的结构化词条（JSON）。
- `anki/` —— 生成的 Anki 卡片。

## 用法

1. 把扫描页放进 `pages/`，按页码命名。
2. OCR 解析，产出 `ocr/`。
3. 生成 `anki/` 卡片，导入 Anki。

> 词条的字段结构、双栏版面、跨页接续的处理规则等实现细节，见 [CLAUDE.md](./CLAUDE.md)。
